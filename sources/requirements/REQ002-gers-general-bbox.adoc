[req-gers-bbox]

[requirement]

====
[%metadata]
name:: Definition of a bounding box
identifier:: req/gers-general-bbox
part:: The bounding box _SHALL_ be in the Coordinate Reference System (CRS) used for a given implementation of the GERS framework.
part:: The bounding box is a rectalinear area that _SHALL_ be defined by values for xmin, xmax, ymin, ymax. Latitude is a decimal number between -90.0 and 90.0 and longitude is a decimal number between -180.0 and 180.0. 
guidance:: This bbox definition used in GERS is different as defined in most OGC, IETF, and ISO Standards. For example, GeoJSON specifies the bbox as min_longitude, min_latitude, max_longitude, max_latitude. 
====
